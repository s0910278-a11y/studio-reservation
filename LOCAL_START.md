# 楽器スタジオ予約システム ローカル起動マニュアル

本システムをローカル（自分のPC）で起動し、ブラウザ上で動作確認するための手順書です。

## 1. 初回のみ行う作業（セットアップ）
一番最初にシステムを動かすとき、または他のパソコンにコピーして初めて動かすときのみ実施します。

1. フォルダ内にある **`setup_once.bat`** をダブルクリックして開きます。
2. 黒い画面が開き、必要な設定が数分かけて自動で行われます。
3. 「セットアップが完了しました！」と表示されたら、エンターキー等を押して画面を閉じます。

## 2. 通常の起動方法
システムを使ったり確認したりする際の、普段の起動方法です。

1. **`start_localhost.bat`** をダブルクリックします。
2. 黒い画面が開き、システムが起動します。この画面は **起動中は絶対に閉じないでください**。
3. 起動後、約5秒ほどで自動的にウェブブラウザが開き、予約システムの画面が表示されます。
   （開かない場合は、ブラウザのアドレスバーに `http://localhost:3001` と入力してください）
4. **終了方法**: システムを使い終わったら、起動しておいた黒い画面（コマンドプロンプト）の右上にある「×」ボタンを押して閉じてください。

## 3. 調子がおかしいとき（強制再起動）
「画面が更新されない」「なぜかエラーが出る」といった場合、バックグラウンドのプロセスが残ってしまっている可能性があります。

1. **`restart_localhost.bat`** をダブルクリックします。
2. 裏で動いているシステムを強制終了し、自動で再度立ち上げ直します。

---

## 起動に失敗する場合の確認箇所

> [!WARNING]
> もし黒い画面にたくさんエラーが出て起動できない場合は、以下を確認してください。

* **アドレスバーなどで「アクセスできません」と出る場合**
  → プログラムの起動が完了していません。黒い画面で「Ready in XXXX ms」と表示されているか確認してください。
* **ポート競合エラーが出る場合 (`Error: listen EADDRINUSE: address already in use :::3001`)**
  → すでにシステムが裏で動いているか、別のアプリがポート 3001 を使っています。`restart_localhost.bat` を実行するか、PCを再起動してください。
* **Prisma などデータベース関連のエラー (`Invalid Prisma Client` など)**
  → なんらかの理由でデータベースが準備されていません。再度 `setup_once.bat` を実行して、エラーが出ないか確認してください。
